# MazeCoinGame

C++ software to solve simple maze puzzle

## Input

```text
13 40
########################################
#i..............m.........#.#...#......#
#.#.#.#.#.##.#.#.######.#.#...#.#.####.#
#.#.#####.##.#.#.....######.##.#..#....s
#.#...#.#.####.#.####........#.#.#.#####
#.######.....##.#......####.##.#.#...#.#
#.############.#.##.#######.##.#.###.#.#
#.#............m..........#.#...m......#
#.##.####.#########.#.#.#.#.###.#.#.##.#
#.#...#...#....#....#.....#.#.#.m.#.##.#
#.#.#######################.#.########.#
#.............#m.......................#
########################################
```


i: entrance

s: exit

m: coin

#: walls

## Output
```text
p1 total steps
p2 steps to the exit
m1 total collected coins
m2 coins in the exit path
x1 y1
x2 y2
...
xn yn
```
